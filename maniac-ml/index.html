PHYS 250 Fall 2018 Computing Platform

JupyterLab-based infrastructure for Computational Physics

Purpose

A computational platform that supports on-demand JupyterLab instances for interactive python development as well as advanced computational resources such as those required for high-level, compute-intensive machine learning applications.

Elements

The platform provides hosted JupyterLab instances for the students in PHYS 250 (Autumn 2018) on GPU resources hosted by the computing center for the ATLAS Experiment (called the Midwest Tier 2). Elements of the SLATE Kubernetes platform and effort from the VC3 project were used in the development of this platform
